> Aprenda programação orientada a objetos neste curso passo a passo.

Ruby é uma linguagem de programação versátil e orientada a objetos, 
criada para proporcionar a felicidade do programador. 
Aprender a programar em Ruby vai permite que você crie uma variedade infinita de programas.

Seguiremos uma abordagem passo a passo, baseada em projetos, para aprender a escrever
programas Ruby bem projetados. 
Ao criar um projeto real com código real do zero, você saberá como escrever seus próprios programas Ruby 
e terá uma compreensão mais profunda de como e por que seguir determinado caminho.

> E ai vamos Criar um programa Ruby completo e testado do zero?

#### CONTEÚDO

##### 1 - AMBIENTE DE PROGRAMAÇÃO RUBY

- Instalando o Ruby no seu sistema operacional favorito
- Executando Ruby usando o shell Ruby interativo (irb) e escrevendo arquivos de programa Ruby
- Usando o sistema de documentação do Ruby para obter ajuda
- Instalando bibliotecas externas do Ruby usando RubyGems
- Solucionando problemas comuns

##### 2 - CONSTRUCTOR RUBY

- Expressões e variáveis
- Numbers, string e symbols
- Loops e expressões condicionais
- Arrays e hashes
- Classes, modules e structs

##### 3 - PROGRAMAÇÃO ORIENTADA A OBJETOS

- Usando classes internas do Ruby
- Definindo suas próprias classes
- Criando objetos únicos
- Dizendo aos objetos o que fazer
- Herança entre classes
- Compartilhando código com mixins

##### 4 - PRINCÍPIOS DE DESIGN EM ORIENTAÇÃO A OBJETOS

- Encapsulamento
- Separando responsabilidades
- Polimorfismo
- Evitando redundâncias - Don't Repeat Yourself (DRY)
- Dizendo a que veio - Tell, Don't Ask

#### 5 - BLOCKS E ITERATORS

- Métodos internos que aceitam blocos
- Yield/Blocks
- Implementando iterators customizados
- Usando blocks em seus programas de maneira efetiva

#### 6 - ORGANIZANDO SEU CÓDIGO RUBY

- Criando uma estrutura de projetos Ruby
- Separando os arquivos de código para facilitar o reuso e testes
- Evitando conflitos de nomeação - Namespacing

#### 7 - INPUT/OUTPUT

- Lendo dados de arquivos
- Gravando dados em arquivos

#### 8 - TESTE UNITÁRIO

- Escrevendo e executando testes de unidade com o RSpec
- Test-driven development (TDD) e o ciclo de red-green-refactor
- Controlando seus testes com stubbing methods
- Refatorando código com segurança

#### 9 - DISTRIBUIÇÃO DE CÓDIGO RUBY

- Convenções do RubyGems
- Escrevendo uma GemSpec
- Construindo uma RubyGem
- Publicando sua gem em um servidor público

#### 10 - O QUE VEM DEPOIS

- Comunidade
- Oportunidades de emprego
