Apesar de parecer complicado, a construção de uma página Web não é física nuclear,
as páginas Web não passam de documentos de texto simples, que listam e 
indexam outros tipos de arquivos, internos ou externos a sua extrutura de pastas.


Para isso não precisamos fazer uso de nenhuma ferramenta drag and drop, basta um editor de texto simples,
como o Notepad do Windows. A diferença é que as páginas Web contêm algumas marcações especiais para determinar
a função de cada elemento dentro do texto. Alguns elementos são marcados como títulos, outros como 
parágrafos. Estas marcações são usadas também para indicar os elos de ligação que levam para outros 
documentos na rede, estes elos são os links. 
Essas marcas são chamadas de tags e estão especificadas dentro da linguagem utilizada para criar as páginas Web, HTML.
